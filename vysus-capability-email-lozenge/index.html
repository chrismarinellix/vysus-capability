How to Create the Animated GIF

  • 1

    Download ScreenToGif
    Free tool from screentogif.com — install and open it.

  • 2

    Click "Recorder"
    A transparent frame will appear. Resize it to fit snugly around your chosen lozenge above.

  • 3

    Record for 2.5 seconds
    Click the red Record button, wait for one full shimmer cycle, then click Stop.

  • 4

    Save as GIF
    In the editor, go to File → Save As and save as .gif format.

Add to Outlook Desktop Signature

  • 1

    Open Signature Settings
    In Outlook: File → Options → Mail → Signatures

  • 2

    Create or Edit a Signature
    Click New or select an existing signature to edit.

  • 3

    Insert the GIF
    Click the Image icon in the editor toolbar and select your saved GIF file.

  • 4

    Add the Link
    Select the GIF image, then click the Hyperlink icon and enter:
    https://vysus-capability.netlify.app/capability.html

  • 5

    Save
    Click OK to save your signature. The animated lozenge will now appear in your emails!

Note: The shimmer animation will play in most email clients. Some older clients may show a static image, which still looks great.
